# Xára!

Projeto desenvolvido com objetivo de aprender/experimentar a biblioteca Chakra UI.

A aplicação é um resumo de uma pesquisa realizada pelo IBGE sobre a frequência dos nomes por década de nascimento, ou seja, você consegue descobrir quantos Lucas, Enzos e Valentinas nasceram.

Os dados são públicos e estão disponíveis (link abaixo).

## Tecnologias utilizadas

* [NextJS](https://nextjs.org)
* [Typescript](https://www.typescriptlang.org)
* [Chakra UI](https://chakra-ui.com)


## Como funciona

1. Digite seu primeiro nome e pronto.
2. Os dados são buscados e computados em uma tabela.


## Iniciando a aplicação

### JS Applications

1. Clone o projeto.
2. Entre na pasta do projeto e execute o seguinte comando:
```sh
  $ yarn
```
3. Após a instalação de todas as dependências, basta execute o seguinte comando:
```sh
  $ yarn dev
```
3. A aplicação ficará disponível na url http://localhost:3000.
